## 2025-05-14 - [Draggable UI for Mobile]
**Learning:** Mobile players in Roblox struggle with fixed UI elements that might block their view or controls. Sliders that only respond to clicks (InputEnded) feel unresponsive on touch devices.
**Action:** Always implement custom dragging logic for main frames and ensure sliders respond to `InputChanged` to allow for smooth dragging and immediate feedback.
